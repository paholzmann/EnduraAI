**Idea: Ultra Route Generator**

A tool where users can input a target region (e.g. Salzburg area), desired distance (e.g. 100 miles), and elevation gain (e.g. 10,000 m), and the system generates a matching running route.

The system should:

* Generate a route (later with GPX export)
* Estimate difficulty (0–100 score + category)
* Analyze elevation profile and steepness
* Optionally adapt to user performance level

Long-term vision:
A personalized endurance planning system that connects route generation, difficulty modeling, and performance prediction to help runners choose optimal routes and races.
